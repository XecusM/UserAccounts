# UserAccounts

UserAccounts Django Project with custom user data, activation email and verfication email

**ATTENTION**
To use this project you have to modify the following variables in settings.py

  * BASE_DIR
  * SECRET_KEY
  * DEBUG
  * ALLOWED_HOSTS
  * DATABASES
  * STATIC_ROOT
  * EMAIL_BACKEND
  * EMAIL_HOST
  * EMAIL_HOST_USER
  * EMAIL_PORT
  * EMAIL_HOST_PASSWORD
  * EMAIL_USE_TLS
  * EMAIL_USE_SSL
  * GOOGLE_RECAPTCHA_SECRET_KEY
